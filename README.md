# 협업준비 과정

## 분할 구조도 (유스케이스 다이어그램)

-  외부 요소와 시스템 간의 상호 작용을 정리해 시스템의 범위를 파악함
-  사용자와 요구 사항과 관리자의 작업 사항 분석을 위한 도구로 사용함
